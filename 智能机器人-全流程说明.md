# 智能机器人 → 智能表格 → 自动分析 → 后续工作

用智能机器人自动往「智能表格」里填数据，系统会**自动分析**，你可以在网页上做查询、导出等后续工作。

---

## 整体流程

```
智能机器人
    ↓ POST /api/smart-table/row（或 /rows）
服务器「智能表格」自动追加一行/多行
    ↓ 服务端立即重算
可分析表格（analyzed）自动更新
    ↓
后续工作：打开网页 → 按角色查、按日期看、导出、设合并规则 等
```

---

## 第一步：让智能机器人自动填充智能表格

在机器人里配置「写数据」的动作为：

- **URL**：`https://你的服务地址/api/smart-table/row`（每写一条就调一次）  
  或 `https://你的服务地址/api/smart-table/rows`（一次写多条）
- **方法**：POST
- **Content-Type**：application/json
- **Body 示例（一条）**：
  ```json
  {
    "date": "2026-03-11",
    "roleName": "角色名",
    "diff": 5000,
    "expStart": 10000,
    "expEnd": 15000
  }
  ```
  也可用中文键：`日期`、`角色名称`、`差值`、`经验值(开)`、`经验值(结束)`。

更详细的字段说明、多行格式、可选密钥，见 **[机器人回调-智能表格.md](./机器人回调-智能表格.md)**。

---

## 第二步：自动分析（无需额外操作）

机器人每写入一行（或一批），服务器会：

1. 把数据追加到内建「智能表格」
2. **立刻**根据整张表重算统计（按日、按角色、总差值等）
3. 更新「可分析表格」

所以**不需要**再点「分析」按钮，分析是自动、实时的。

---

## 第三步：后续工作（在网页上完成）

分析结果更新后，你可以：

| 操作 | 说明 |
|------|------|
| **按角色查** | 在网页搜索框输入角色名，看该角色所有记录、按日经验、总和 |
| **按日期看** | 看某一天各角色的经验与差值 |
| **合并规则** | 管理员在「合并规则」里把同人不同名合并，统计会按规范名汇总 |
| **导出** | 需要时导出为 Excel（.xlsx）做报表或备份 |
| **API 查询** | 其他系统可调 `/api/role?name=xxx` 等接口取数据 |

打开 **网站首页** 即可做以上操作，数据会随机器人写入而实时更新。

---

## 小结

- **智能机器人**：按文档调 `POST /api/smart-table/row`（或 `/rows`）自动填充智能表格。
- **分析**：由服务端自动完成，无需人工点分析。
- **后续工作**：在网页上查询、合并规则、导出，或通过 API 给别的系统用。

把机器人的「写表」地址设成你的服务地址 + `/api/smart-table/row`，即可走通「自动填充 → 自动分析 → 后续工作」全流程。
